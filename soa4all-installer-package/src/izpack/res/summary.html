What's next?

Launching

Go to the '${INSTALL_PATH}' directory and launch the 'soa4all.(sh|bat)' script, it will launch the application server with the services you selected.

Additional configuration

If you installed the Sesame module, you will need to create repositories you defined in dependent modules.
Please go to http://${installer.tomcat.hostname}:${installer.tomcat.http}/openrdf-sesame and create the following repositories :
LIST(installer.sesame.repo.)

Playing

Once started, have a look to http://${installer.tomcat.hostname}:${installer.tomcat.http} which runs the software modules you selected.

NOTE : If you installed the Dashboard module, you should have a look to it at http://${installer.tomcat.hostname}:${installer.tomcat.http}/dashboard.
