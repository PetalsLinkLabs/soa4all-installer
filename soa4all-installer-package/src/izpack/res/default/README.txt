SOA4All - http://soa4all.eu

This folder has been created from the SOA4All installer program.
This file explains how to launch the different modules which are composing the SOA4All platform.

A/ Starting the platform

./startup.sh

B/ Stopping the platform

./shutdown.sh

C/ Web applications
The Web applications are automatically deployed into the local Tomcat instance at installation time. 
The local Apache Tomcat instance is available under $PATH/apache-tomcat where $PATH is the base installation folder.

D/ Distributed Service Bus
This binary is installed by another binary. Please refer to its README file for more information on how to use it.

E/ Uninstalling the platform

./uninstall.sh
